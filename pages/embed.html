Tag <embed>



Definição e Uso

  • A <embed> tag define um contêiner para um recurso externo, como uma página da Web, uma imagem, um media player ou um aplicativo de plug-in.

Atributos

Atributo Valor Descrição
height pixels Especifica a altura do conteúdo incorporado
src URL Especifica o link do arquivo incorporado
type media_type Especifica o tipo de mídia do conteúdo incorporado
width pixels Especifica a largura o conteúdo incorporado
