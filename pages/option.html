Tag <option>



Definição e Uso

  • A tag <option> define uma opção em uma lista de seleção.

    Elementos <option> vão dentro de um elemento <select>, <optgroup> ou <datalist>.

Atributos

Atributo Valor Descrição
disabled disabled Especifica que uma opção deve ser desabilitada
label text Especifica um rotulo mais curto para uma opção
selected selected Especifica que uma opção deve ser pré-selecionada quando a página for carregada
value text Especifica o valor a ser enviado para um servidor
