Tag <del>



Definição e Uso

  • A tag <del> define o texto que foi excluído de um documento. Os navegadores geralmente riscam uma linha através do texto excluído.


Atributos

Atributo Valor Descrição
cite URL Especifica uma URL para um documento que explica o motivo pelo qual o texto foi excluído/alterado.
datetime YYYY-MM-DDThh:mm:ssTZD Especifica a data e hora de quando o texto foi excluído/alterado.
