LISTA DE REFERÊNCIAS


  • <applet>

    A tag foi usada no HTML4 para definir um applet embutido (Plug-in).

  • <col>

    A tag especifica propriedades de coluna para cada coluna dentro de um elemento <colgroup>.

  • <del>

    A tag define o texto que foi excluído de um documento.

  • <embed>

    A tag define um contêiner para um recurso externo.

  • <header>

    A tag representa um contêiner para conteúdo introdutório.

  • <img>

    A tag é usada para incorporar uma imagem em uma página HTML.

  • <label>

    A tag define um rótulo para vários elementos.

  • <meta>

    A tag define metadados sobre um documento HTML.

  • <option>

    A tag define uma opção em uma lista selecionada.

  • <p>

    A tag define um parágrafo.

  • <s>

    A tag especifica o texto que não está mais correto, preciso ou relevante.

  • <span>

    A tag é um contâiner embutido usado para marcar uma parte de um texto ou uma parte de um documento.

  • <summary>

    A tag define uma direção visível para o elemento <details>.

  • <track>

    A tag especifica faixas de texto para elementos <audio> ou <video>.

  • <u>

    A tag representa algum texto não articulado e com estilo diferente do normal.
