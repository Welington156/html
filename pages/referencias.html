Lista de Referências

  • col

    A tag especifica propriedades de coluna para cada coluna dentro de um elemento 'colgroup'.

  • option

    A tag define uma opção em uma lista selecionada.

  • summary

    A tag 'summary' em HTML é usada para definir um resumo para o elemento > < detalhes.

  • track

    A tag especifica faixas de texto para elementos de audio ou vídeo.

  • s

    A tag especifica texto que não está mais correto, preciso ou relevante. O texto será exibido com uma linha através dele.
