Tag <track>



Definição e Uso

  • A tag especifica faixas de texto para elementos <audio> ou <video>.

  • Este elemento é usado para especificar legendas, arquivos de legendas ou outros arquivos contendo texto, que devem ser visíveis quando a mídia estiver sendo reproduzida.

  • As faixas são formatadas no formato WebVTT (arquivos.vtt).

Atributos

Atributo Valor Descrição
default default Especifica que a faixa deve ser habilitada se as preferências do usuário não indicarem que outra faixa seria mais apropriada
kind captions
chapters
descriptions
metadata
subtitles		 Especifica o tipo de faixa de texto
label text Especifica o título da faixa de texto
src URL Requeridos. Especifica o URL do arquivo de faixa
srclang language_code Especifica o idioma dos dados de texto da faixa (obrigatório se kind="legendas")
