Tag HTML <label>

Definição e Uso

A tag <label> define um rótulo para vários elementos:

  • <input type="checkbox">
  • <input type="color">
  • <input type="data">
  • <input type="datetime-local">
  • <input type="email">
  • <input type="arquivo">
  • <input type="mês">
  • <input type="número">
  • <input type="senha">
  • <input type="radio">
  • <input type="intervalo>
  • <input type="pesquisa">
  • <input type="tel">
  • <input type="texto">
  • <input type="time">
  • <input type="url">
  • <input type="semana">
  • <metro>
  • <progresso>
  • <selecione>
  • <textarea>

Dicas e Notas

Dica: O atributo for de <label> deve ser igual ao atributo id do elemento relacionado para vinculá- los. Um rótulo também pode ser vinculado a um elemento colocando o elemento dentro do elemento <label>.

Atributos

Atributo Valor Descrição
for element_id Especifica o id do elemento de formulário ao qual o rótulo deve ser vinculado.
form form_id Especifica a qual formulário o rótulo pertence.
