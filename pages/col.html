COL

Definição

A tag especifica propriedades de coluna para cada coluna dentro de um elemento 'colgroup'. A tag é útil para aplicar estilos a colunas inteiras, em vez de repetir os estilos para cada célula, para cada linha.

Atributos

Atributo Valor Descrição
span number Especifica o número de colunas que um elemento 'col' deve abranger
